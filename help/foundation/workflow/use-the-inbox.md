# 使用AEM中的收件箱

AEM Inbox可促进协作，并可视化任务和工作流。 收件箱的日历视图在基于时间的视图中跟踪项目和任务，以便更轻松地管理和计划项目。 借助日历视图，用户可以全面了解当前正在进行、计划、即将到期和已过期的项目和任务。
