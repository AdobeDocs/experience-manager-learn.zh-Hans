# 使用工作流编辑器

工作流支持Experience Manager中的业务流程管理，用于内容的自动处理，并有助于需要人为决策的治理和流程。 工作流是通过工作流模型定义的，工作流模型由一系列步骤组成，并在[!UICONTROL 工作流模型编辑器]中创建和管理。
