# Experience Manager的人工智能

Experience Manager中的![AI](./assets/overview/hero.png)

Experience Manager as a Cloud Service提供高级AI功能，以增强内容管理、简化工作流并改善用户体验。 通过集成AI支持的功能(如AEM AI Assistant和AEM AI Agent)，用户可以自动执行任务、获得见解并优化内容交付。
