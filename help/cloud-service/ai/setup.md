# 在AEM中设置人工智能

要在AEM中使用AEM AI助手和代理，您需要在Adobe Experience Manager环境中设置访问权限。 本指南将指导您完成启用这些AI支持的功能的必要步骤。 您还可以在操场环境中试验AI助手。

## AEM AI 助手

了解如何使用[Adobe Admin Console](https://adminconsole.adobe.com)在Adobe Experience Manager中配置对AI助理的访问权限。 此步骤对于使用户能够从Assistant获取响应并利用自动票证创建是必需的。

>[!VIDEO](https://video.tv.adobe.com/v/3474066/?learn=on&enablevpops)

### AEM中的代理

要在AEM中使用代理，必须先配置[AEM AI助手](#aem-ai-assistant)访问权限。 AEM中的代理必须由Adobe配置。 要请求访问AEM中的代理，请联系您的Adobe代表或[Adobe支持](https://experienceleague.adobe.com/en/support)。

## 工作区

如果您无权访问或无法在AEM中设置AEM AI助手或代理，则仍可在游乐场环境中探索其功能。

AEM提供了一个游乐场环境，用户可以在其中试验AI助手的功能，包括AEM中的代理，而不影响其生产环境。 这允许用户在安全设置中熟悉AI助手的特性和功能。

![AEM游乐场](./assets/setup/playground.png)

要创建游乐场，请导航到[Experience Hub](https://experience.adobe.com/#/@AdobeOrg/experiencemanager/)，然后在&#x200B;**产品公告**&#x200B;面板中选择&#x200B;**创建游乐场**。 游乐场设置完成后，您将收到一封电子邮件，其中包含访问游乐场的详细信息。
