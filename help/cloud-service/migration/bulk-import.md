# 使用批量导入

通过AEMas a Cloud Service中的批量导入工具，管理员能够以安全、高效的方式从云存储(Azure Blob Storage或Amazon S3)批量导入资产。

>[!VIDEO](https://video.tv.adobe.com/v/329680/?quality=12&learn=on)

## 计划批量导入

批量导入支持按计划执行配置，包括：

+ 在定义的日期和时间执行一次
+ 每小时、每天或每周执行一次期间执行
