# 使用AdobeCloud Manager — 程序

Cloud Manager **程序** 代表支持业务计划逻辑集的AEM环境集，通常对应于购买的服务水平协议(SLA)。 例如，一个程序可能代表支持全球公共网站的AEM资源，而另一个程序代表内部中央DAM。
