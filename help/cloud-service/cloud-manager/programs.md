# 使用Adobe云管理器-项目

Cloud Manager **项目**&#x200B;表示支持业务计划逻辑集的AEM环境集，通常对应于购买的服务级别协议(SLA)。 例如，一个项目可以代表AEM资源以支持全球公共网站，而另一个项目则代表内部的Central DAM。
