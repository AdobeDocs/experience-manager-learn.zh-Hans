# 使用Adobe Cloud Manager -项目

Cloud Manager **项目**&#x200B;表示支持业务计划逻辑集的AEM环境集，通常对应于购买的服务级别协议(SLA)。 例如，一个项目可以表示支持全球公共网站的AEM资源，而另一个项目则表示内部的中央DAM。
