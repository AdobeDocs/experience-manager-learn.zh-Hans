# 使用AdobeCloud Manager — 程序

Cloud Manager **程序**&#x200B;表示支持业务计划逻辑集的AEM环境集，通常与购买的服务水平协议(SLA)相对应。 例如，一个程序可能代表支持全球公共网站的AEM资源，而另一个程序代表内部中央DAM。
