# 创建Adobe I/O项目

1. 创建集成的第一步是创建[Adobe ID](https://account.adobe.com/)
1. 转到[Adobe开发人员控制台](https://console.adobe.io/home)
1. 使用步骤1中创建的Adobe ID登录
1. 创建新项目
