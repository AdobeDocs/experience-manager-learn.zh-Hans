# 创建Adobe I/O项目

1. 创建集成的第一步是创建[Adobe ID](https://account.adobe.com/)
1. 转到[Adobe开发者控制台](https://console.adobe.io/home)
1. 使用在步骤1中创建的Adobe ID登录
1. 创建新项目

请按照视频创建您的Adobe I/O项目。
