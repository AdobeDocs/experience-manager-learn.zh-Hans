# 创建主题

主题包含组件和面板的样式详细信息，您可以在不同表单上重用主题。 样式包括诸如背景颜色、状态颜色、透明度、对齐方式和大小等属性。 将主题应用于表单时，指定的样式会反映在表单的相应组件上。
