# 服务凭据

与AEM作为Cloud Service的集成必须能够安全地对AEM进行身份验证。 AEM Developer Console会生成服务凭据，外部应用程序、系统和服务使用这些凭据，以编程方式通过HTTP与AEM创作或发布服务进行交互。
