# 简介

Azure存储帐户包含所有Azure存储数据对象，包括Blob、文件共享、队列、表和磁盘。 存储帐户为您的Azure存储数据提供了唯一的命名空间，可从世界上的任何位置通过HTTP或HTTPS访问该命名空间。 您的存储帐户中的数据是持久的、高可用的、安全的，并且可大规模扩展的。
