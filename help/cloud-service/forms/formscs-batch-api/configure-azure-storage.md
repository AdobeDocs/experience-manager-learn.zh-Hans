# 简介

Azure存储帐户包含所有Azure存储数据对象，包括Blob、文件共享、队列、表和磁盘。 存储帐户为您的Azure存储数据提供了一个唯一的命名空间，可通过HTTP或HTTPS从全球任何地方访问这些数据。 您的存储帐户中的数据是持久的、高度可用的、安全的，并且可扩展性非常大。
