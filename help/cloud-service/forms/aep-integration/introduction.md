# 将AEM Forms与Adobe Experience Platform(AEP)集成

>[!NOTE]
>
>将自适应Forms (AEM Forms)与Adobe Experience Platform (AEP)连接的功能属于抢先访问计划。请将您的工作地址中的电子邮件发送至aem-forms-ea@adobe.com以请求获取访问权限。

您可以将AEM Forms与Adobe Experience Platform连接起来，以转换您的表单体验。 这种强大的集成使各类组织能够利用实时客户配置文件提供个性化的表单体验，简化向Experience Platform提交AEM Forms数据的过程，并在Adobe生态系统中创建统一的客户记录。

本教程将指导您完成在AEM Forms与AEP之间创建集成所涉及的步骤
