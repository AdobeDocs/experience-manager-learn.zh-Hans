# 创建连接的应用程序

连接的应用程序是一个框架，它允许外部应用程序使用API和标准协议（例如SAML、OAuth和OpenID Connect）与Salesforce集成。 连接的应用程序使用这些协议来验证、授权并提供外部应用程序的单点登录(SSO)。
