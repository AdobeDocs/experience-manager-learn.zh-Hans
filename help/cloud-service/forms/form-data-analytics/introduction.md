# 使用Adobe Analytics报告表单数据字段值和表单字段验证错误

了解如何使用标记和Adobe Analytics在自适应表单上实施分析。 此示例将指导您完成配置和实施步骤，以生成有关访客如何与您的表单交互的深入报告。

## 前提条件

为了充分利用本教程，建议您满足以下先决条件：

* AEM Forms CS的一些经验
* 访问Adobe标记
* 访问Adobe Analytics



本教程使用在AEM Forms中构建的简单自适应表单，并测量表单提交的居留值状态以及生成验证错误的字段。
