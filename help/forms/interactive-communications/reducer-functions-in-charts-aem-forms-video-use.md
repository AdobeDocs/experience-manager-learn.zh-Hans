# 在AEM Forms中使用Reducer函数 — 图表{#using-reducer-functions-in-aem-forms-charts}

Reducer函数用于聚合或分组图表中显示的值。


例如，我们可以使用Reducer函数来显示每个类别的事务处理数，或显示下面示例数据中每个类别的最高费用

```json
[{
"category": "Groceries",
"amount": 10
},
{
"category": "Groceries",
"amount": 20
},
{
"category": "Travel",
"amount": 100
},
{
"category": "Gas",
"amount": 500
},
{
"category": "Gas",
"amount": 80
},
{
"category": "Groceries",
"amount": 30
},
{
"category": "Travel",
"amount": 200
}
]
```
