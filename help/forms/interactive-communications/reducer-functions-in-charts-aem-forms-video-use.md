# 在AEM Forms中使用减速器函数——图表{#using-reducer-functions-in-aem-forms-charts}

Reducer函数用于聚合或组图表中显示的值。


例如，我们可以使用类别函数显示每个类别的事务处理数，或根据以下示例数据显示每个客户的最高费用

```json
[{
"category": "Groceries",
"amount": 10
},
{
"category": "Groceries",
"amount": 20
},
{
"category": "Travel",
"amount": 100
},
{
"category": "Gas",
"amount": 500
},
{
"category": "Gas",
"amount": 80
},
{
"category": "Groceries",
"amount": 30
},
{
"category": "Travel",
"amount": 200
}
]
```
