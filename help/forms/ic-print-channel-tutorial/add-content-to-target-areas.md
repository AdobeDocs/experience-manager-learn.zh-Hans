# 将文本和图像内容添加到打印渠道文档

在本部分中，我们将向布局(xdp)中定义的目标区域添加图像和文本内容。

既然您已经创作了将包含文档名称和地址的第一个文档片段，我们需要将收件人片段添加到打印渠道文档。

您可以向打印渠道文档添加各种类型的内容，如文档片段、图像、布局片段等。

要向打印渠道文档添加内容，请观看以下视频。

视频将指导您完成向打印目标文档中的适当渠道区域添加内容所涉及的步骤。
