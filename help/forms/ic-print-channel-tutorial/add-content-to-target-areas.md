# 将文本和图像内容添加到打印渠道文档

在本部分中，我们将向版面(xdp)中定义的目标区域添加图像和文本内容。

现在，您已创作第一个包含收件人姓名和地址的文档片段，我们需要将该文档片段添加到打印渠道文档。

您可以将各种类型的内容（如文档片段、图像、布局片段等）添加到打印渠道文档中。
