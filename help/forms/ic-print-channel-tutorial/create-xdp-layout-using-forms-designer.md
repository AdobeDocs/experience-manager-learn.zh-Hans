# 使用Forms设计器{#create-layout-using-forms-designer}创建布局

在本部分，我们将看到使用AEM Forms设计器创建的布局文件。

每个印刷渠道文档都将基于使用AEM Forms设计器工具创建的模板。 通常，您将在模板中创建用于保存目标片段、图像或布局片段的章节或文档区域。 理想情况下，布局将不定义任何内容。 这会将布局与推荐的做法内容分离。
