# AEM Forms，含JSON模式和数据

多部分教程，用于指导您完成创建带有JSON模式的自适应表单和查询提交数据所涉及的步骤。

要学习本教程，建议您安装以下软件

* AEM 6.3或更高版本（包含适当的forms addon包）

* MySQL 8数据库

* MySQL 8工作台

此外，建议您具备一定的AEM和AEM Forms经验。 您还应熟悉安装MySQL数据库和MySQL工作台
