# AEM Forms，含JSON模式和数据

多部分教程，指导您逐步完成使用JSON模式创建自适应表单和查询提交数据所涉及的步骤。

要学习本教程，建议您安装以下软件

* AEM 6.3或更高版本（包含适当的表单addon包）

* MySQL 8数据库

* MySQL 8工作台

还建议您具备一定的AEM和AEM Forms经验。 您还应熟悉安装MySQL DB和MySQL工作台
