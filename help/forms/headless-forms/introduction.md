# 嵌入无头自适应表单

本文将介绍在Spa网站中嵌入无头自适应表单的基础知识。 本文假定您现有的单页应用程序，并且希望使用核心组件嵌入在AEM Forms 6.5.16中创作的自适应表单。
通过在单页应用程序中包含表单，用户可以无缝地输入和提交数据，而无需刷新页面。 这增强了应用程序的交互性和效率。
