# AEM Workflow中的变量

从AEM 6.5开始，您现在可以在AEM Workflow中创建和使用变量。 在此功能之前，工作流作者通常使用工作流元数据映射创建键。 这意味着编写一些自定义代码来创建和填充这些密钥。 借助这个新的功能工作流，作者现在可以轻松创建变量并在工作流中的任意位置使用这些变量。

变量可以是以下类型

* 长整型
* 双精度型
* 布尔值
* 字符串
* 日期
* JSON
* XML
* 数组列表
* FormDataModel
* 文档
