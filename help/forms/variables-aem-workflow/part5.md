# AEM Workflow中的规则编辑器

“或 — 拆分”组件用于在AEM Workflow中进行工艺路线。 在AEM 6.5版本之前，您必须编写ECMA或自定义代码以启用条件路由。 在AEM 6.5中，我们现在能够使用规则编辑器创建将决定要执行的路由的业务规则。
