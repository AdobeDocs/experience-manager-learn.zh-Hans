# AEM工作流中的文档类型变量


文档类型的变量用于存储记录、自适应表单附件的文档。 然后，可在“发送电子邮件”组件中使用这些变量通过电子邮件发送附件。

此视频将突出显示如何使用文档类型变量存储生成的DoR，然后在AEM工作流的发送电子邮件步骤中使用文档变量。
