# 向帐户余额面板添加表{#adding-table-to-account-balance-panel}

在本部分中，我们将向“帐户余额”面板添加一个表。

登录AEM Forms，然后导航到Adobe Experience Manager > Forms > Forms和文档。

打开401KStatement文件夹。

在编辑模式下打开401KStatement。

点按“帐户余额”目标区域，然后单击“+”图标以调出“插入组件”对话框。

选择表以插入表组件。
