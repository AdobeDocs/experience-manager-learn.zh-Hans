# 将文本和图像内容添加到Web渠道文档

在本部分中，我们将向标题面板添加图像和文本内容。

既然您已经创作了将包含文档名称和地址的第一个文档片段，我们需要将收件人片段添加到Web渠道文档。

您可以向Web文档文档添加各种类型的内容，如渠道片段、图像、布局片段等。

要向Web渠道文档添加内容，请观看以下视频。

该视频将指导您完成向Web目标文档中的适当渠道区域添加内容所涉及的步骤。
