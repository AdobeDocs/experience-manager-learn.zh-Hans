# 简介

在本教程中，您将学习如何实施一个简单的用例，即使用提交的数据和附件预填基于&#x200B;**核心组件**&#x200B;的表单。

## 先决条件

* 使用包含核心组件的AEM Forms的体验
* 具有构建和部署AEM项目方面的经验。
* 访问AEM Forms 6.5.19或更高版本。

要在AEM Forms中实施上述用例，您需要满足以下条件

* AEM Forms 6.5.19或更高版本实例
* [Azure门户帐户](https://portal.azure.com/)
