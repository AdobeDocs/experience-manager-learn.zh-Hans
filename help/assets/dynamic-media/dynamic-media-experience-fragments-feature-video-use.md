# 在AEM Assets Dynamic Media中使用体验片段{#using-experience-fragments-with-aem-assets-dynamic-media}

AEM Assets Dynamic Media通过交互式媒体查看器将资产热点与体验片段集成。 营销人员可以使用文本、图像和链接来创作体验片段，以填充自定义弹出窗口，并将该弹出窗口与横幅、轮播或视频上的热点相关联。
