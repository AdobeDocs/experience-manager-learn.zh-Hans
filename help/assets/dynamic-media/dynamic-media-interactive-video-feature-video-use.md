# 在AEM Dynamic Media中使用交互式视频{#using-interactive-video-with-aem-dynamic-media}

AEM Assets现在能够创建交互式视频，用户可以在其中将视频拆分为较小的区段，或合并小的区段以创建较大的区段，从而允许用户关联行动动手。
