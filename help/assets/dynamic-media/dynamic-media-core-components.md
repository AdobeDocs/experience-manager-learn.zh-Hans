# 将Dynamic Media与AEM Sites核心组件{#dynamic-media-sites-core-components}结合使用

图像组件是Experience Manager核心组件的一部分，它内置了对Dynamic Media的支持。 图像组件允许内容作者在AEM Sites页面上使用Dynamic Media的功能，如图像预设、智能裁剪和图像修饰符。
