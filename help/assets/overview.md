# AEM Assets 视频和教程 {#overview}

{{edge-delivery-services}}

Adobe Experience Manager(AEM) Assets 是 AEM Platform 上的一款数字资产管理工具，它允许用户在基于 Web 的存储库中创建、管理和共享其数字资产（图像、视频、文档和音频剪辑）。 本用户指南包含了有关 AEM Assets 的众多特性和功能的视频和教程。
