# 资产搜索 {#asset-search}

通过Adobe资产链接面板，创意用户可以使用关键字搜索AEM Assets中存储的资产，或在特定位置下执行搜索。 除了基本的搜索选项之外，Adobe资产链接还允许您使用资产在AEM Assets中搜索视觉上类似的结果。 视觉搜索由Adobe Sensei的机器学习技术提供支持，可显示与目标图像相关的美学特征结果。
