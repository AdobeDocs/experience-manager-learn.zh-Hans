# 资产搜索 {#asset-search}

借助Adobe Asset Link面板，创意用户可以使用关键字搜索AEM Assets中存储的资源，或在特定位置下执行搜索。 除了基本的搜索选项之外，Adobe Asset Link还允许您使用资源在AEM Assets中搜索视觉上类似的结果。 视觉搜索由带有Adobe Sensei的机器学习提供技术支持，可显示具有与目标图像相关的美学特征的结果。
