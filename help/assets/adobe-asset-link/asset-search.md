# 资产搜索 {#asset-search}

利用“Adobe资产链接”面板，创意用户可以使用关键字搜索存储在AEM Assets的资产，或在特定位置下执行搜索。 除了基本的搜索选项之外，Adobe资产链接还允许您使用资产在AEM Assets内搜索视觉效果相似的结果。 视觉搜索功能以Adobe Sensei的机器学习为后盾，可显示与您的目标图像相关的审美特征结果。
