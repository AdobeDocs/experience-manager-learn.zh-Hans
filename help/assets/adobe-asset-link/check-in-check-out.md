# 登记注销 {#check-in-check-out}

Adobe资产可让您直接从您正在处理的创意应用程序中查看AEM Assets，并立即开始进行编辑。 在您的Creative Cloud应用程序中编辑完资产后，您可以使用正确的版本控制和注释轻松将其存回AEM Assets。
