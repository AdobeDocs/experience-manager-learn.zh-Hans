# 签入和签出 {#check-in-check-out}

Adobe资源允许您直接从正在处理的创意应用程序中签出AEM Assets，并立即开始进行编辑。 在Creative Cloud应用程序中编辑完资源后，您可以轻松地将资源签回到AEM Assets，其中包含正确的版本控制和注释。
