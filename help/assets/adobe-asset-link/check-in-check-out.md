# 签入签出{#check-in-check-out}

Adobe资产允许您直接从正在处理的创意应用程序中签出AEM Assets，并且可以立即开始进行编辑。 在AEM Assets应用程序中编辑完资产后，您可以使用正确的版本控制和注释轻松将资产签回Creative Cloud。
