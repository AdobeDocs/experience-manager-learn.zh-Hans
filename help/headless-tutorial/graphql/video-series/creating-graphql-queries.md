# 创建GraphQL查询

了解如何创建GraphQL查询以从Adobe Experience Manager(AEM)返回内容，以及如何使用GraphQL工具快速测试、优化和调试查询。

在此视频中，您将：

+ 了解GraphQL语言背后的强大功能
+ 了解如何查询内容片段和单个内容片段的列表
+ 了解如何过滤和请求特定数据属性
+ 了解如何安装和使用GraphiQL工具

>
