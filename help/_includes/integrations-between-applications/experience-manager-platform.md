{{experience-manager-description}}

{{experience-platform-description}}

将AEM与Experience Platform集成有几个好处，包括：

+ **数据摄取**：从各种来源摄取和统一数据，以实现整体客户视图。
+ **客户配置文件**：使用行为、事务性和人口统计数据创建统一的用户档案。
+ **激活和集成**：与Adobe产品集成，以实现实时数据激活和洞察。
