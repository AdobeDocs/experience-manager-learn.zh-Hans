# 裁剪、调整图像和缩放目标 {#crop-adjusted-zoom-targets}

动态媒体经典主控图像概念的主要优势之一是您可以将图像资产重新用于多种用途。 通常，您必须为每个图像创建单独的裁剪版本才能显示详细信息或显示色板。 使用Dynamic Media Classic时，您可以对单个主控执行相同的任务，并将这些裁剪后的版本另存为新的物理文件或另存为不占用存储空间的虚拟衍生文件。

在本教程的本节结尾，您将了解如何：

- 在Dynamic Media Classic中裁剪图像，并另存为新的主控文件或虚拟图像。 [了解更多](https://docs.adobe.com/help/en/dynamic-media-classic/using/master-files/cropping-image.html).
- 保存虚拟调整后的图像，并将其用作主控资源。 [了解更多](https://docs.adobe.com/content/help/en/dynamic-media-classic/using/master-files/adjusting-image.html).
- 在图像上创建缩放目标以展示其高光。 [了解更多](https://docs.adobe.com/content/help/en/dynamic-media-classic/using/zoom/creating-zoom-targets-guided-zoom.html).

## 裁剪

Dynamic Media Classic在UI中提供一些图像编辑工具，包括裁剪工具。 您可能希望在Dynamic Media Classic中裁切主控图像，原因有多。 例如：

- 您无权访问原始文件。 您希望以不同的裁剪或长宽比显示图像，但您的计算机上没有原始文件，或者正在家中工作。 在这种情况下，您可以进入Dynamic Media Classic，找到图像，裁剪并保存图像，或将其另存为新版本。
- 删除多余的空白。 照片中的空白太多，产品看起来很小。 您希望您的缩略图图像能够尽可能地填充画布。
- 要创建调整后的图像，请创建不占用磁盘空间的图像的虚拟副本。 一些公司有业务规则，要求他们保留同一图像的不同副本，但名称不同。 或者，您希望同一图像的裁剪和未裁剪版本。
- 从源图像制作新图像。 例如，您可能希望创建色板或主图像的详细信息。 您可以在Adobe Photoshop执行此操作并单独上传，或在Dynamic Media Classic中使用裁剪工具。

>[!NOTE]
>
>以下关于裁剪的讨论中的所有URL仅用于说明性目的；它们不是实时链接。

### 使用裁剪工具

您可以从资产的详细信息页面或通过单击编辑按钮来访问Dynamic Media Classic中的裁剪 **工具** 。 您可以使用该工具通过两种方式进行裁剪：

- 默认裁剪模式，在该模式下，您可以拖动裁剪窗口的手柄或在“大小”框中键入值。 了解如何手 [动裁剪](https://docs.adobe.com/content/help/en/dynamic-media-classic/using/master-files/cropping-image.html#select-an-area-to-crop)。
- 修剪。 通过计算与图像不匹配的像素数，使用它删除图像周围的额外空白。 了解如何通 [过修剪进行裁剪](https://docs.adobe.com/content/help/en/dynamic-media-classic/using/master-files/cropping-image.html#crop-to-remove-white-space-around-an-image)。

### _手动裁剪_

保存手动裁剪的版本时，图像会被永久裁剪；Dynamic Media Classic实际上通过添加内部URL修饰符来裁剪图像来隐藏像素。 发布时，图像被裁剪的所有人都会看到，但您可以返回到裁剪编辑器并在稍后删除裁剪。

然后，您可以选择另存为新主控图像还是另存为主控的附加视图。 新主控是占用存储空间的新物理文件（如TIFF或JPEG）。 另一个视图是不占用服务器空间的虚拟映像。 我们不建议您选择“替换原稿”，因为这将覆盖您的主控并使裁剪成为永久裁剪。 如果您另存为新主控或其他视图，则必须选择新的资产ID。 与其他资产ID一样，它必须是Dynamic Media Classic中的唯一名称。

### _裁切裁剪_

如果您上传的图像主题周围有太多空白（额外的画布），则调整大小后，它在Web上的显示效果会小得多。 对于150像素或更小的缩略图图像尤其如此——照片的拍摄对象可能会在周围的所有额外空间中丢失。

比较同一图像的这两个版本。

![图像](assets/crop-adjusted-zoom-targets/trim-cropping.jpg)

通过删除产品周围的额外空间，右边的图像变得更加突出。 可以使用裁剪工具一次修剪一个图像，也可以在上传时以批处理形式运行。 如果您希望将所有图像一致地裁剪到主主题的边界，则建议以批处理方式运行。 将裁切裁切到定界框——图像周围的矩形。

>[!NOTE]
>
>裁切不会在图像周围创建透明度。 为此，您需要在图像上嵌入一个剪切路径，并使用“从剪 **辑路径创建蒙版** ”上传选项。
>
>此外，要在使用“保存”选项裁剪图像后将其恢复到原始状态，请在“裁剪编辑器”屏 **幕中显** 示该图像，然后选择“重置 **”按钮** 。

### _上传时裁剪_

如前所述，您还可以在上传图像时选择裁剪图像。 要在上传时使用修剪裁剪，请单击“作 **业选项** ”按钮，在“裁剪选项”下，选 **择修剪**。

Dynamic Media Classic将记住此选项，以便下次上传。 您可能希望它为此上传裁切图像，但可能不希望每次上传都裁切图像。 另一个选项是设置一个特殊的计划FTP上传作业，并将裁剪选项放在那里。 这样，您只会在需要裁切图像时运行作业。

>[!IMPORTANT]
>
>如果您为上传设置了裁剪，Dynamic Media Classic将会放置一个cookie，以便下次记住该设置。 作为最佳实践，在下次上 **传前单击“重置为公司默认值** ”按钮，以清除上次上传后留下的任何裁剪选项；否则，您可能会意外裁剪下一批图像。

### 按URL裁剪

尽管在Dynamic Media Classic中不明显，但您也可以仅通过URL进行裁剪（甚至可以向图像预设添加裁剪）。

每次使用裁剪工具时，您都会在底部的字段中看到URL值。 您可以使用这些值并将它们作为URL修饰符直接应用于图像。

![图像](assets/crop-adjusted-zoom-targets/cropping-by-url.png)_裁剪编辑器底部的裁剪命令修饰符_

![图像](assets/crop-adjusted-zoom-targets/uncropped-cropped.png)

由于使用裁切裁剪裁剪时必须根据图像计算大小，因此无法通过URL自动进行裁剪。 裁切裁剪只能在上传时运行，或通过一次应用一个图像来运行。

### _在图像预设中裁剪_

图像预设中有一个字段，您可以在该字段中添加其他图像服务命令。 要向图像预设添加与上面相同的裁剪，请编辑您的预设并粘贴或在URL修饰符字段中键入值，然后保存并发布。

![图](assets/crop-adjusted-zoom-targets/cropping-in-image-preset.jpg)像&#x200B;_向图像预设的URL修饰符中添加裁剪命令（或任何命令）。_

该裁剪现在将成为该图像预设的一部分，并且每次使用时会自动应用。 当然，此方法取决于所有需要相同裁剪量的图像。 如果图像不是以相同的方式拍摄，则此方法将不适合您。

## 调整后的图像

使用裁剪工具时，您可以选择“另存 **为主控的附加视图**”。 保存后，将创建一种新的Dynamic Media Classic资产——调整后的图像。 调整后的图像也称为导数，是虚拟图像。 它实际上根本不是一幅图像；它是对物理主控映像的数据库引用（如别名或快捷方式）。

### 真实形象能站起来吗`?`

您能否判断哪个是主控，哪个是调整后的图像？

![图像](assets/crop-adjusted-zoom-targets/real-image-stand-up.png)

在查看Dynamic Media Classic并查看SBR_MAIN2的“调整后的图像”资产类型之前，您将无从得知。

调整后的图像不使用磁盘空间，因为它仅作为行项存在于数据库中。 它还与原始资产永久绑定；如果删除原始图像，则还会删除调整后的图像。 它可以由整个未裁剪的图像组成，也可以只是图像（裁剪）的一部分。

![图像](assets/crop-adjusted-zoom-targets/adjusted-image.png)

您通常使用裁剪工具创建调整后的图像；但是，也可以使用其他图像编辑器（调整和锐化工具）创建它们。

调整后的图像需要唯一的资产ID。 发布后（您必须像发布任何其他资产一样进行发布），这些资产将充当任何其他图像，并通过其资产ID在URL上调用。 在“详细信息”页面上，您可以在“构建和衍生项”选项卡下视图与主控图像关 **联的已调整图** 像。

## 缩放目标

缩放目标也可在图 **像的** “编 **辑** ”菜单和“详细信息”页上找到。 它们允许您设置“热点”以突出缩放图像的特定推销功能。 缩放查看器可提供图像顶部的详细信息以及您创建的简短标签，而不是通过裁切大型主控来创建单独的图像。

![图像](assets/crop-adjusted-zoom-targets/arm-with-watch.jpg)

由于缩放目标本质上是一个销售功能，并且需要了解产品的销售点，因此通常由公司的销售或产品团队中的人员创建。

该过程非常简单——单击该功能，为它指定一个描述性名称并保存。 目标可以从一幅图像复制到另一幅图像（如果它们相似），但该过程是手动的。 Dynamic Media Classic无法自动创建缩放目标，因为每个图像都不同，并具有不同的功能。

决定是否使用缩放目标的另一个因素是您选择查看器。 并非所有查看器类型都可显示缩放目标（例如，弹出查看器不支持它们）。

了解如何创 [建缩放目标](https://docs.adobe.com/content/help/en/dynamic-media-classic/using/zoom/creating-zoom-targets-guided-zoom.html#creating-and-editing-zoom-targets)。

![图像](assets/crop-adjusted-zoom-targets/zoom-targets.jpg)

### 使用缩放目标工具

以下是在Dynamic Media Classic中创建目标的工作流。

1. 浏览到图像，单击“编 **辑** ”按钮，然后选 **择“缩放目标**”。
2. 将加载缩放目标编辑器。 您会在中间看到您的图像，顶部有一些按钮，右边有一个空的目标面板。 在左下角，您将看到一个已选择的查看器预设。 默认为“Zoom1-Guided”。
3. 用鼠标移动红色框并单击以创建新目标。

   - 红框是目标区。 当用户单击该目标时，它将放大到框内的区域。
   - 目标大小由查看器预设内的视图大小决定。 这决定了主缩放图像的大小。 请参 _阅下面的设置视图_&#x200B;大小。

4. 您将看到您刚刚创建的目标变为蓝色，在右侧将显示该目标的缩略图版本以及默认名称“目标-0”。
5. 要重命名目标，请单击其缩略图，键 **入新名**&#x200B;称，然后单 **击Enter** 或 **Tab** —如果您只是单击“离开”，您的姓名将不会保存。
6. 选中目标时，框周围将有绿色虚线，您可以调整其大小并移动它。 拖动角调整大小，或拖动目标框移动。

   - 这将在默认的自定义缩放查看器中加载图像。 确保查看器预设支持缩放目标-通常，所有带有“向导”字样的标准预设都设计为可与缩放目标一起使用。 要使用目标，请将鼠标悬停在目标缩略图（或热点图标）上以查看标签，然后单击它以查看查看器放大到该功能。
   - 与在Dynamic Media Classic中执行的所有其他工作一样，您必须发布内容才能在Web上实时显示缩放目标。 如果您使用的查看器支持目标，它们将立即显示（一旦清除缓存）。 但是，如果您未使用启用缩放目标的查看器，它们将保持隐藏。

      ![图像](assets/crop-adjusted-zoom-targets/zoom-target-green-box.jpg)

7. 此外，如果需要删除目标，请通过单击其缩略图将其选中，然后按“删 **除目标** ”按钮或按键盘上的DELETE键。
8. 继续单击以添加新目标、重命名和／或在添加后调整大小。
9. 完成后，单击“保 **存** ”按钮，然后 **预览**。

### 在缩放查看器预设中设置视图大小

让我们来谈一谈缩放目标的大小来自何处。 在缩放查看器的查看器预设中，有一个称为视图大小的设置。 视图大小是查看器中缩放图像的大小。 它与舞台大小不同，舞台大小是查看器的总大小，包括UI组件和图稿。

创建新目标时，其大小和长宽比由视图大小派生。 例如，如果视图大小为200 x 200，则您将只能制作方形目标，最大缩放区域为200像素。 您的目标可以大于200像素，但始终为正方形。 但这也意味着缩放查看器中的图像只有200像素——缩放目标的大小与查看器的大小直接相关。 因此，在设置目标之前，您首先决定查看器设计。

但是，默认情况下视图大小为空（设置为0 x 0），因为主视图图像的大小是动态的，并会根据舞台的大小自动派生。 问题是，如果您未在预设中明确设置视图大小，缩放目标工具将不知道制作目标的大小。

加载缩放目标工具时，视图大小会显示在预设名称的旁边。 比较内置的“缩放1”向导式预设和自定义ZT_AUTHORING预设之间的视图大小。

![图像](assets/crop-adjusted-zoom-targets/view-size.jpg)

您可以看到内置预设的大小为900 x 550，这意味着目标永远不可能变得比这大。 这可能太大——如果您有2000像素的图像，您只能发出一个至少900像素的跨度功能。 用户可以手动进一步缩放，但您无法近距离引导他们。 将视图大小设置为350 x 350可使目标几乎放大或放大。 但是，如果您希望查看器中有更大的缩放图像，则需要创建新的预设，因为您的预设锁定在350像素。

### 创建或编辑支持缩放目标的查看器预设

要设置视图大小，请创建或编辑支持缩放目标的查看器预设。

1. 在查看器预设中，转到缩放 **设置选** 项。
2. 设置宽度和高度。
3. 保存预设，然后关闭。 如果要在您的实时站点上使用该预设，您以后也必须进行发布。
4. 进入缩放目标工具，然后在左下角选择您编辑的预设。 您会立即看到新视图大小反映在目标中。
