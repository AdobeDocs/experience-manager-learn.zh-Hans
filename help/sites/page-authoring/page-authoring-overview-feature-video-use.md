# 編寫頁面 {#authoring-page-aem-sites}

瞭解如何使用網站編輯器的UI在Adobe Experience Manager Sites中編寫頁面。

>[!VIDEO](https://video.tv.adobe.com/v/31828?quality=12&learn=on)

本影片涵蓋下列功能或特色：

* 根據範本建立頁面
* 為頁面匯入不同的撰寫模式
   * 编辑
   * 布局
   * 开发人员
   * 时间扭曲
* 頁面編輯器功能
   * 元件尋找器
   * 资产查找器
* 將內容新增至現有元件
* 將新元件新增至頁面
* 頁面屬性
   * 发布页面
   * 取消發佈頁面
   * 编辑模板
   * 以发布的形式查看
