# Edge Delivery Services和Universal Editor开发人员教程

![Edge Delivery Services和Universal Editor开发人员教程](./assets/0-overview/hero.png)

在本教程中，您将学习构建一个AEM网站的基础知识，该网站将强大的创作与通用编辑器结合使用，并使用Edge Delivery Services进行超快交付。 到最后，您将基本了解如何创建新项目、设置本地开发环境和构建新块。

## 项目设置

了解如何在AEM as a Cloud Service中创建代码项目并配置新站点。 通过此设置，可以使用通用编辑器实现无缝开发，以便通过Edge Delivery Services创建内容和快速交付内容。
